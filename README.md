## An interactive exploration of McLeod's Pitch Detection Method

A Julia (Pluto.jl) implementation of the 2005 paper titled ["A Smarter Way To Find Pitch" by Philip McLeod et al](https://citeseerx.ist.psu.edu/document?repid=rep1&type=pdf&doi=60dd4c01f687858a5fbf6c021920c56247bcf2db#page=1.74).

Please see the [introductory blog post](https://samyak.me/post/mcleod/) for more information.

[Live link](https://mcleod.samyak.me/) (it's interactive if the server is up. If not, you can still access a static version of it).
